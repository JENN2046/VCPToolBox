# PhotoStudioDeliveryTasks

Generates delivery-stage tasks for an existing `photo_studio` project without changing the rest of the task set.

## Command

- `create_delivery_tasks`

## Input

```json
{
  "project_id": "proj_ab12cd34",
  "override_existing": false,
  "delivery_mode": "online gallery",
  "delivery_deadline": "2026-05-25"
}
```

## Notes

- Allowed project states: `reviewing`, `delivered`, `completed`
- Generates exactly 4 delivery-stage tasks per run
- Re-running without `override_existing=true` skips the existing delivery-stage task set
- Only replaces delivery-stage tasks previously generated by this plugin
