# VCP 交互联通钉钉 - 快速入门指南

> 5 分钟快速开始，将 VCP 接入钉钉平台

---

## 前置要求

- [ ] Node.js 18+ 已安装
- [ ] 钉钉企业账号（或个人账号）
- [ ] VCP 服务器已启动

---

## 第一步：创建钉钉应用 (5 分钟)

1. 访问 [钉钉开放平台](https://open.dingtalk.com/)
2. 登录并创建**企业内部应用**
3. 获取 `AppKey` 和 `AppSecret`
4. 配置事件订阅（机器人消息）

### 应用配置清单

| 配置项 | 说明 |
|--------|------|
| 应用类型 | 企业内部应用 |
| 应用能力 | 机器人 |
| 事件订阅 | 接收消息回调 |

---

## 第二步：配置适配器 (2 分钟)

1. 复制环境变量文件：
```bash
cd Plugin/vcp-dingtalk-adapter
cp .env.example .env
```

2. 编辑 `.env` 文件：
```env
DING_APP_KEY=your_app_key
DING_APP_SECRET=your_app_secret
VCP_USE_CHANNEL_BRIDGE=true
VCP_CHANNEL_BRIDGE_URL=http://127.0.0.1:6005/internal/channel-hub/events
VCP_API_KEY=your_vcp_api_key
```

---

## 第三步：启动适配器 (1 分钟)

```bash
cd Plugin/vcp-dingtalk-adapter
npm install
npm start
```

看到以下日志表示启动成功：
```
============================================================
✅ VCP DingTalk Adapter is ready!
📡 Waiting for DingTalk messages...
============================================================
```

---

## 第四步：测试消息 (1 分钟)

在钉钉中向机器人发送消息：
```
你好
```

如果收到 AI 回复，说明集成成功！

---

## 常见问题

### Q: 收不到消息？
A: 检查钉钉事件订阅配置，确保回调地址正确

### Q: 回复失败？
A: 检查 VCP 服务器是否运行在 6005 端口

### Q: 如何修改 AI 人格？
A: 修改 `VCP_AGENT_NAME` 环境变量
