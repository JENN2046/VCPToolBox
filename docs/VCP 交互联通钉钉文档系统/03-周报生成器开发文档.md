## 功能特性

### 1. generate_from_logs - 从工作记录生成周报
- 读取指定日期范围的工作记录
- 调用 AI 生成结构化周报
- 支持多种格式输出 (Markdown/Text/JSON)

### 2. generate_from_diary - 从日记系统生成周报
- 从 VCP 日记系统读取记录
- 自动识别本周时间范围
- 生成周报摘要

### 3. export_to_table - 导出到钉钉 AI 表格
- 将生成的周报导出到钉钉表格
- 支持自定义表格 UUID
- 返回记录 ID 和状态

## 调用示例

### 生成周报
```
<<<[TOOL_REQUEST]>>>
tool_name:「始」WeeklyReportGenerator「末」,
action:「始」generate_from_logs「末」,
week_start:「始」2026-03-24「末」,
week_end:「始」2026-03-30「末」
<<<[END_TOOL_REQUEST]>>>
```

### 导出周报
```
<<<[TOOL_REQUEST]>>>
tool_name:「始」WeeklyReportGenerator「末」,
action:「始」export_to_table「末」,
content:「始」# 第 13 周周报\n\n本周完成...\n\n## 经验教训\n- ...\n\n## 下周计划\n- ...「末」,
summary:「始」第 13 周工作总结「末」
<<<[END_TOOL_REQUEST]>>>
```
