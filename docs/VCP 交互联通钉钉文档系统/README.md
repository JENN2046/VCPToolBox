## 概述

VCP 交互联通钉钉文档系统包含以下核心能力：

### 1. 钉钉渠道接入
- 企业内部应用集成
- 机器人消息收发
- 事件订阅回调

### 2. 工作日志自动化
- 随时记录工作内容
- 定时生成日报/周报
- 钉钉 AI 表格写入
- 周报自动生成

### 3. VCP 核心能力
- ChannelHub B2 协议
- 插件化扩展
- 多模态消息处理

## 快速开始

```bash
# 1. 配置环境变量
cd Plugin/vcp-dingtalk-adapter
cp .env.example .env

# 2. 安装依赖
npm install

# 3. 启动适配器
npm start
```
