### 安全化改动

GPTImageGen 是外部 API 调用插件，且会写入本地图片文件。因此本次没有按作者线启用态直接吸收，而是按稳定线安全态处理：

- 默认不加载插件、不注入工具描述、不调用外部 API
- 不提交真实 `Plugin/GPTImageGen/config.env`
- 无 `OPENAI_API_KEY` 时插件离线退出，不发起 API 请求
- 本地图片输入限制在项目 `image/` 目录内
- 阻止本机、链路本地和私有网段图片 URL 输入
- 下载远程图片时按 4MB 上限提前中断
- 生成图片目录 `image/gptimagegen/` 不进入 Git 或 Docker build context
