## VCP Overview

### Core Concepts
- **RAGDiaryPlugin**: Message preprocessor plugin for memory retrieval
- **MetaThinkingManager**: Recursive inference chain manager
- **SemanticGroupManager**: Semantic group management
- **ContextVectorManager**: Context vector mapping

### Key Features
- **Semantic Group Enhancement**: Query vector enhancement through keyword matching
- **Meta-Logic Chunks**: Modular storage of reasoning patterns
- **Super-Dynamic Recursive Fusion**: Progressive vector enhancement
